# Share Chat Builder Stories 181-200

These stories exercise the `/s` right-side Chat pane as a project-building agent. They intentionally span static sites, apps, bots, APIs, and operational tooling so the agent needs general multi-file creation, safe configuration, and concise verification habits rather than prompt-specific shortcuts.

## 181. Marketing Page
As a solo founder, I want to ask Chat to create a polished marketing page for a new analytics product so I can publish a credible first impression from one prompt.

Acceptance:
- Creates a landing page with hero, proof, feature, pricing, testimonial, FAQ, and CTA sections.
- Includes responsive styling and non-placeholder copy.
- Keeps the project runnable with package/config files when needed.

## 182. Restaurant Website
As a restaurant owner, I want Chat to create a reservation-ready restaurant website so customers can see menu highlights, opening hours, events, and booking CTAs.

Acceptance:

- Uses warm visual language suited to hospitality.
- Keeps copy concrete enough to publish after light editing.
